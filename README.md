# Clínica Veterinária - Projeto de HTML

Este projeto foi desenvolvido como parte do desafio **"Criando seu Primeiro Site Completo com HTML"**.  
O objetivo foi criar um site funcional de uma clínica veterinária, aplicando todos os conceitos do módulo de HTML.

## Funcionalidades e Conteúdos

O site inclui:

- **Formulários:** formulário de contato para clientes enviarem mensagens.  
- **Estruturação e formatação de texto:** títulos, parágrafos e seções bem organizadas.  
- **Mídias:** imagens dos animais.  
- **Tabelas:** tabela de horários de atendimento.  
- **Menu de navegação:**  
  - Página Principal  
  - Sobre a Clínica  
  - Horário de Atendimento  
  - Contato  

O projeto demonstra a utilização prática do HTML, seguindo todas as instruções do módulo.
